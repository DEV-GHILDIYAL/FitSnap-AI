# FitSnap AI — Next.js Application

This is the core Next.js application. See the [root README](../README.md) for full documentation.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Script | Description |
|--------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Production build |
| `npm run build:lambda` | Build for AWS Lambda (standalone + asset copy) |
| `npm run deploy` | Build + deploy to AWS Lambda via Serverless |
| `npm run lint` | Run ESLint |
